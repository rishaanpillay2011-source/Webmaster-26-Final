Code & Repository

Pathfinders is built on a modern, lightweight, and incredibly fast tech stack. Our goal is to ensure the platform remains fully accessible, performant, and easy to maintain for future contributors.

We believe in open-source development and transparency. This project relies entirely on raw web standards — HTML, CSS, and Vanilla JavaScript — avoiding heavy frameworks to deliver an optimal experience on any device, from desktop to low-bandwidth mobile.

HTML & CSS Architecture

The foundation of Pathfinders is built on semantic HTML5. This approach ensures maximum accessibility for screen readers and search engines. We strictly avoid deeply nested, meaningless <div> structures, preferring elements like <header>, <main>, <section>, and <article> to clearly outline the document's structure.

Styling is handled entirely by vanilla CSS, utilizing modern features such as CSS Grid, Flexbox, and CSS Custom Properties (Variables). By defining a central design token system in our :root, we can seamlessly switch between the Light and Dark themes. The dark mode logic alters these variables globally, ensuring high-contrast readability without duplicating stylesheets. The layout is fluid and responsive, relying on clamp(), min(), and max() functions along with CSS media queries to dynamically adjust typography and spacing from ultra-wide desktops down to mobile devices.

JavaScript for Functionality

To keep the application fast and free of unnecessary bloat, we chose Vanilla JavaScript over heavy frontend frameworks like React or Vue. Our functionality is modularized into discrete files within the js/functionality/ directory.

This includes the theme toggle initialization, dynamic navigation rendering, and form handling. A significant portion of the logic is dedicated to the Resource Directory and Map pages. For the directory, JavaScript fetches and filters the JSON resource data in real-time, instantly updating the DOM without page reloads. For the map, we integrate Leaflet.js to plot interactive markers based on community resource coordinates, providing spatial awareness for users seeking localized help.

JavaScript for Animations

The engaging, cinematic feel of the Pathfinders website is powered by the GreenSock Animation Platform (GSAP). While our custom animation logic is neatly organized in the js/animations/ folder, the core GSAP library files themselves are located in the gsap/ directory.

Note: The gsap/ folder contains external library code provided by GreenSock and was not authored by the Pathfinders development team.

Instead of relying on generic CSS transitions tied to scroll events, ScrollTrigger calculates precise scroll progress to scrub animations. This drives the parallax wave effects, the sticky image stacking section on the homepage, and the staggered reveal of UI elements as they enter the viewport. We also use the SplitText plugin to separate headings into individual words and characters, allowing us to choreograph intricate, randomized fade-in effects that give the typography a vibrant, organic feel.

Open Source Repository

Want to contribute, fork the project, or explore the raw code? The entire codebase is available on GitHub. We welcome pull requests, bug reports, and feature suggestions from the community. Whether you're fixing a typo in the CSS or proposing a new directory feature, your contributions help make Pathfinders better for everyone.
